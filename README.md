# Quantum

Welcome to **Quantum**! 

## Contributions

#### Follow these steps to set up the project locally:

1. **Clone the repository**:   
```bash
git clone --recursive https://github.com/AustinLynes/Quantum.git
```
Navigate to the project directory:

```bash
cd Quantum
```
Run the Buld file to generate project files and current build of the engine

```bash
.\build.bat -generate   # generates the vs solution files.
.\build.bat -build      # builds the current version of the project.
.\build.bat             # (vs2022 recommended) builds and generates the current project.  
```
## 2. Change Branch and Make Changes
create a branch so you can make changes.
```bash
git branch -b 'branch-name'
```
commit your changes.

```bash
git add *changed files*
git commit -m "commit description, please be descriptive"
git push origin branchname 
```

## 3. Create a Pull Request! 
Open a Pull Request on GitHub and describe your changes.
